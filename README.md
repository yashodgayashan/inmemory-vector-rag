# Simple RAG System

## Set Up

Please follow the following steps to setup

1. Create virtual environment

```python
python3 -m venv .venv
```

2. Activate virtual environment

```python
source .venv/bin/activate
```

For Windows

```python
.venv\Scripts\activate
```

3. Install requirements

```python
pip install -r requirements.txt
```

4. Deactivate

```python
deactivate
```
